# Трекер завдань

Виконання лабораторної роботи передбачає виконання прийомів роботи з системою контролю версій Git.

- створення відділеного репозиторію;
- клонування відділеного репозиторію;
- виконання комітів;
- створення гілок та перемикання між ними;
- виконання комітів у різних гілках;
- аутентифікація на GitHub за допомогою токена та через SSH;
- виванження локального репозиторію на GitHub.

У практичній роботі №1 було:

1. створено локальний репозиторій
2. створено відділений репозиторій на GitHub
3. зроблено коміти у локальному репозиторії
4. виванважено усі коміти з локального у відділений репозиторій

## Використані команди Git

У практичній роботі №1 було використано такі команди Git:

- git init
- git status
- git add
- git commit
- git remote add
- git push
- git pull
- git branch
